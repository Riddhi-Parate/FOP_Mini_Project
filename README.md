# FOP_Mini_Project
C Around Us ( Poster and project on digital banking system and alarm clock system )

This repository contains two projects

  Banking System (C + HTML/CSS/JS)
  
  Alarm Clock System (C + HTML/CSS/JS)

These projects demonstrate how core logic written in C programming can be extended into user-friendly web applications.

Banking System

🔹 Description

A console-based banking application built in C that allows users to manage accounts, perform transactions, and track history.
This logic is later extended into a web-based interface using HTML, CSS, and JavaScript.

🔹 Features

Create Account

Login System (Password Protected)

Deposit Money

Withdraw Money

Transfer Money

Transaction History

Account Details Management

Password & PIN Encryption

File Handling using CSV

🔹 Tech Used

C Programming

File Handling

HTML, CSS, JavaScript (for UI version)

Alarm Clock System

🔹 Description

A simple alarm clock system built using C and extended into a web-based version for better usability.

🔹 Features

Set Alarm Time

Real-Time Clock Display

Alarm Notification

User-Friendly Interface (Web version)

Simple and Efficient Design

🔹 Tech Used

C Programming

Time Functions

HTML, CSS, JavaScript
